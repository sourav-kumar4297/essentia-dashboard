# essentia · Sales & Client Acquisition Portal (v1)

Basic workflow portal for **essentia group** — Growth → Acquisition → Closing.

**Visual system** (matches [essentiaenvironments.com](https://essentiaenvironments.com/)): Classic Black `#000000`, white logo at 20px in the header, Lato 300 labels at 13px, Lato light metrics at 12px, white text.

## Demo login

| Field    | Value         |
|----------|---------------|
| Username | `admin`       |
| Password | `password123` |

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or the port shown in the terminal).

## v1 scope (basic)

1. **Lead Inbox** — qualify, assign, follow-up, handoff  
2. **Consultation** — discovery capture  
3. **Company Profile** — draft / preview / send (demo)  
4. **Design Fee Proposal** — rate card, live totals, accept  
5. **Closing** — order + handoff package  

Data stays in the browser (local storage). **No HubSpot or other CRM** in v1 — that comes in a later phase.
